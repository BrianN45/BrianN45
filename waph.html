Course Details

In this course, students will learn the basics behind web development. This will include learning how to create a front-end and a back-end. You will also create a database to connect it to your website. There will be multiple exercises to reinforce knowledge and hackathons to teach about vulnerabilities on a website.

Topics Covered in This Course

  • Front-End Development with HTML, CSS, and JavaScript
  • Back-End Development with SQL, PHP, and Databases
  • Version Control with Git
  • Website Vulnerabilities
  • Common Practices in Web Development

Course Goals, Objectives, and Outcomes

At the end of this course, students should be able to:

  • Create websites using the latest technologies.
  • Vulnerabilities in web applications and how to avoid them
  • Keep track of projects using Git
  • Secure their own website by testing out vulnerabilities
  • Gain practical knowledge that can be applied outside of this course
